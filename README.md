# Astro Blog ShokaX

这是 [Hexo Theme ShokaX](https://github.com/theme-shoka-x/hexo-theme-shokaX) 在 Astro 上的重构版本，使用 Astro + Svelte 5 + UnoCSS 技术栈

在 Astro 版本的 ShokaX 上，得益于底层技术的进步，我们实现了：
- 视图过渡动画（SPA体验）
- 更好的动画与视觉效果
- 基于 Typescript 的现代统一配置系统
